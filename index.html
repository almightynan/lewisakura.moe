...and nice to meet you! I'm Lewis (but you can also call me Lewi), a software engineer — primarily backend — and I exist! I currently mostly write projects for my own personal use, as part of teams, or as commissions for freelance.

More?

Nice to see your curiosity! I was 18 the last time this site was updated, and I have a huge interest in technology. Whilst software engineering is my passion and my major, I've dabbled in network engineering and cybersecurity. They're really fun topics, so I'm very passionate about all three!

I really like designing pretty websites, but actually implementing them is a problem for me. I've unfortunately had to throw away projects because of it mainly because they were web design heavy or they depended on public usage. However, some projects I've been able to keep because of being in teams! I really like working in teams of people since it helps me distribute the workload and I can focus on my strengths, which is usually engineering backend technology.

I'm also a game developer, primarily working on the Roblox platform. Despite the stigma around it, I think it's a great platform for learning and I've had a lot of fun working on it. I've also used Unity, Unreal, and Godot.

Here's some quickfire fun facts about this website:
  • The CSS is a derivative of the excellent sakura.css framework. I wrote the CSS myself, but the majority is based on this and sometimes is even verbatim. It's a wonderful framework, you should check it out!
  • I wrote this in approximately 2 hours.
  • Shiki (the code highlighter on this site) is mainly a build-time tool, but I wrote some very awful JavaScript to get it to work at runtime. Please do not look at the source, it's very overengineered and I'll throw it away someday.
  • Speaking of build-time tools, there are none here. I manually wrote all the HTML because I'm a masochist and I haven't futureproofed this at all.
  • The monospace font is Iosevka (specifically the web variant), and everything else is Inter, unless of course none of it loaded.
